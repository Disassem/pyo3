{% meta %} tags: [versions, python] title: Вышел python 2.5-beta2 {% endmeta %} {% mark body %} 11 июля вышла вторая бета Python-2.5. Впереди: первого августа rc1 и восьмого - релиз. Ждем-с. Со времени же первой беты, разработчики исправили множество ошибок (см. список изменений). Добавлена новая функция sys._current_frames(). Разработчики уверяют, что это очень полезная функция для отслеживания взаимных блокировок и других проблем; аналогичная технология используется в популярном расширении для Zope DeadlockDebugger. Также по умолчанию убраны предупреждения при импорте директорий, не содержащих __init__.py. {% endmark %}
