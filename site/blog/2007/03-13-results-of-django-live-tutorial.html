{% meta %} tags: [mine, events, django] title: Результаты Django Live Tutorial {% endmeta %} {% mark body %} Сегодня прошел семинар Django Live Tutorial. Дело происходило на факультете компьютерных наук ОмГУ. Что из себя представлял данный семинар: я по слайдам сделал небольшую вводную по Django (в основном по определениям), потом вживую показал как создается Django-приложение (где-то на полтора часа). Делал действительно вживую, т.е. заранее код не писал, лишь прикинул модели да набросал план рассказа. Было где-то человек десять. Естественно, большинство - студенты ФКН. По результатам могу сказать: формат "live tutorial" показался мне удачным. Планирую на этом не останавливаться. Из моментов, заслуживающих критики, могу отметить непродуманность плана доклада. Сам заметил, что "скакал" с одного места на другое. Анонсы мероприятия есть на форумах PyDev и FKN, там же и некоторые отзывы участников. Код и презентация - на code.google.com. {% endmark %}
